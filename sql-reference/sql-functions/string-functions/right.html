# right

# Description

# Syntax

VARCHAR RIGHT (VARCHAR STR)

It returns the right part of a string of specified length, length is char length not the byte size.

# example

mysql> select right("Hello doris",5);
+-------------------------+
| right('Hello doris', 5) |
+-------------------------+
| doris                   |
+-------------------------+

# keyword

RIGHT
